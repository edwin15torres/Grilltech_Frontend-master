Proyecto Ionic 2.0
El proyecto se enfocó en modernizar un sistema de software existente mediante la implementación de Docker Compose para la gestión de contenedores y la migración del motor de base de datos de SQLite a PostgreSQL. Modernización de la arquitecura de datos y arquitectura de Software "Aplicación de escritorio"
El proyecto de rediseño de la arquitectura de datos para la aplicación de apuestas en Python, con la integración de SQLite y la aplicación de Scrum como marco ágil, resultó en un sistema más eficiente, escalable y fácil de mantener. La adopción de buenas prácticas de desarrollo y gestión de proyectos permitió alcanzar los objetivos del proyecto de manera exitosa y satisfactoria.

Este proyecto se centró en la implementación y aplicación de modelos autoregresivos en el contexto de machine learning. Los modelos autoregresivos son técnicas poderosas para predecir series temporales y patrones de datos secuenciales.
